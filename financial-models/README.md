**Objective**  
To estimate the intrinsic value of Netflix, Inc. using a combination of Discounted Cash Flow (DCF) analysis and comparable multiples valuation.

**Methodology**
- Five-year financial projections
- Revenue, operating cost, capex, and depreciation forecasting
- Weighted Average Cost of Capital (WACC) estimation
- Discounted Cash Flow (DCF) valuation with terminal value
- Scenario analysis (Base, Best, and Worst cases)
- Comparable company multiples valuation

**Key Outputs**
- Intrinsic value per share estimation
- Sensitivity analysis on WACC and long-term growth assumptions
- Investment recommendation based on valuation range

**Tools Used**
- Microsoft Excel (financial modeling and valuation)
- PDF export for presentation
